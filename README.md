# quick-mark package

Convert lightweight markup to an HTML form.

##### Example Markup

<pre>
  &lt;
  First Name?
  Last Name?
  &gt;
  blue&lt;
  Suffix...
  Go.
  HINT: Click "Go" when ready. 
  &gt;
</pre>
